# 📱 Avaliação Formadora 2 - Color Change App

Projeto desenvolvido em Ionic e Angular para a disciplina de Desenvolvimento de Aplicativos Mobile da Unisuam.

## 🎯 Objetivo do Projeto
O aplicativo consiste em duas telas principais:
1. **Cartão de Identificação:** Uma tela de apresentação pessoal contendo dados acadêmicos.
2. **Misturador RGB (Color Change):** Uma interface interativa utilizando `ion-range` para capturar valores de Vermelho, Verde e Azul (0 a 255) e aplicá-los dinamicamente aos botões e ao fundo da tela utilizando o conceito de *Two-Way Data Binding* (`[(ngModel)]`).

## 🚀 Tecnologias Utilizadas
* Ionic 7/8
* Angular (Standalone Components)
* TypeScript
* SCSS
* HTML5
